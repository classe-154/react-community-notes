## 📝 1.1 Introduzione a JavaScript

### 💡 1.1.1 Cos'è JavaScript e a cosa serve

Immagina di voler creare un sito web. Con l'HTML crei la struttura (i testi, i titoli, i bottoni), con il CSS decidi lo stile (i colori, lo sfondo, i font). Ma il sito, finché usi solo questi due strumenti, è fermo, statico.

JavaScript è il linguaggio di programmazione che trasforma quel sito statico in qualcosa di vivo e interattivo. È il "motore" che permette al sito di reagire a quello che fa l'utente.

A cosa serve concretamente? Serve a far apparire un messaggio quando clicchi su un bottone, a controllare se l'email inserita in un modulo è corretta, a creare un timer che fa il conto alla rovescia o a far muovere gli elementi sulla pagina. Si usa ogni volta che una pagina web deve fare qualcosa di attivo invece di stare semplicemente ferma a farsi leggere.

### 💻 1.1.2 Esempio base

Questo è un esempio semplicissimo per vedere JavaScript in azione. Immaginiamo di voler inviare un saluto sulla schermata del computer.

```JavaScript
// Creiamo un messaggio di saluto e lo mostriamo sul computer
let messaggio = "Ciao! Benvenuto nel mondo di JavaScript!";
alert(messaggio);
```

🔍 **Spiegazione dell'esempio:**

- `let messaggio = ...`: La parola let serve a creare una scatola (chiamata in gergo variabile) a cui diamo il nome messaggio. Dentro questa scatola mettiamo il testo "Ciao! Benvenuto nel mondo di JavaScript!".
    
- `alert(messaggio);`: Il comando alert dice al browser (il programma che usi per navigare su internet) di prendere quello che c'è dentro la scatola messaggio e mostrarlo all'utente dentro una finestrina pop-up che appare in alto nella pagina.
    

### 🛠️ 1.1.3 Esempio pratico

Ora facciamo un esempio più vicino a una situazione reale: un piccolo sistema che dà il benvenuto a un utente registrato, simulando il controllo del suo nome.

```JavaScript
// Memorizziamo il nome dell'utente
let nomeUtente = "Luca";

// Creiamo il testo del saluto unendo il testo fisso al nome inserito
let salutoPersonalizzato = "Benvenuto nel tuo profilo, " + nomeUtente + "!";

// Mostriamo il risultato nella console del browser (un'area di lavoro per sviluppatori)
console.log(salutoPersonalizzato);
```

🔍 **Spiegazione dell'esempio:**

- Quando il programma gira, per prima cosa crea la scatola nomeUtente e ci salva dentro la parola "Luca". Se l'utente cambiasse nome (ad esempio "Sofia"), basterebbe cambiare il valore in questa scatola.
    
- Nella riga successiva, usiamo il simbolo + non per fare una somma matematica, ma per unire i testi. JavaScript prende "Benvenuto nel tuo profilo, ", ci attacca il contenuto di nomeUtente (cioè "Luca") e infine aggiunge il punto esclamativo. Salva tutto questo testo unito nella nuova scatola salutoPersonalizzato.
    
- Il comando `console.log(...)` prende il contenuto della scatola e lo stampa in una zona speciale del browser chiamata Console (uno strumento utile per noi che programmiamo per verificare che tutto funzioni). L'utente vedrà la scritta scritta: Benvenuto nel tuo profilo, Luca!.
    

### ⚠️ 1.1.4 Errori comuni

- **Dimenticare le virgolette per i testi:** Scrivere let nome = Luca; senza virgolette è un errore. JavaScript penserà che Luca sia il nome di un'altra scatola e, non trovandola, bloccherà il programma con un errore. I testi vanno sempre racchiusi tra virgolette " " o apici ' '.
    
- **Sbagliare le maiuscole e le minuscole:** JavaScript è molto fiscale. Se crei una scatola chiamata nomeUtente (con la U maiuscola) e poi provi a usarla scrivendo nomeutente (con la u minuscola), il programma ti dirà che quella scatola non esiste.
    
- **Confondere il nome della scatola con il suo contenuto:** Se scrivi `console.log("nomeUtente");` con le virgolette, sul computer apparirà letteralmente la parola nomeUtente. Se vuoi vedere cosa c'è dentro la scatola, devi scriverlo senza virgolette: `console.log(nomeUtente);`.

🚀 **Key Takeaways del Giorno**

- **Interattività:** L'HTML crea la struttura della pagina, il CSS la rende bella, ma JavaScript è l'unico che la rende attiva e dinamica.
    
- **Variabili come scatole:** Usiamo let per memorizzare i dati (come testi o numeri) dentro a dei contenitori con un nome, così da poterli riutilizzare quando serve.
    
- **Lettura passo passo:** JavaScript esegue le istruzioni in ordine, dall'alto verso il basso, una riga alla volta.

| **Stringa**           | Sequenza immutabile di caratteri codificati, utilizzata per rappresentare dati testuali all'interno del codice source.               | Il termine da nerd per dire "un pezzo di testo".
